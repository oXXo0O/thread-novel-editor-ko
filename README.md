# 스레 형식 소설 에디터

2ch 익명 게시판 스레드 형식의 소설을 편하게 작성하고 투고용으로 내보낼 수 있는 웹 에디터입니다.

**→ [사이트 바로가기](https://oXXo0O.github.io/thread-maker-ko)**

---

## 기능

- **레스 작성** — 번호·이름·날짜 형식으로 레스 추가, 편집, 삭제, 순서 변경
- **고정닉 목록** — 자주 쓰는 닉네임을 메모와 함께 저장, 클릭 한 번으로 입력란에 자동 입력
- **특수기호 팔레트** — 【】〔〕※◆ 등 자주 쓰는 기호 빠르게 삽입, 커스텀 기호 추가 가능
- **미리보기** — 실제 게시판처럼 보이는 프리뷰
- **내보내기** — 2ch 스타일 / 일반 텍스트 형식으로 변환 후 클립보드 복사
- **날짜 설정** — 표시 안함 / 자동(현재 시간) / 고정값 선택 가능, 레스별 개별 수정 가능
- **로컬 저장** — 브라우저를 닫아도 작업 내용 자동 유지 (localStorage)
- **파일 저장/불러오기** — JSON 파일로 내보내고 다른 기기에서 불러오기 가능

## 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl + Enter` | 레스 추가 |
| `Ctrl + S` | 저장 |

## 기기 간 이동

모바일 → PC 등 다른 기기로 작업을 옮길 때는 헤더의 **📤 파일 저장** 버튼으로 `.json` 파일을 내보낸 뒤, 다른 기기에서 **📥 파일 불러오기** 로 복원하면 됩니다.
